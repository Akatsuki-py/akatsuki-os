# Akatsuki OS

## Abstract

CUIの自作OSです。

まだ開発中です。

## Features

このOSには以下の特徴があります。今後増えていく予定です。

- Grub2 bootloader
- PIT
- GDT
- Keyboard Interrupt
- Multiprocessing
- Memory Management(not paging)

## Usage

### Virtual Boxなどを使う場合

```
$ ./iso.sh
```

### qemuを使う場合

```
$ ./qemu.sh
```
